# Metal.onl

This repository contains the source code for the Metal.onl links website.  
It is open-source and community-editable.

## About

Metal.onl is a curated links website maintained in this repo.  
It includes HTML and CSS that power the website's pages.

## Repository Contents

- `index.html` – Main landing page
- `projects.html` – Projects/links page
- `styles.css` – Site styling
- `script.js` – Interactive behavior
- `img/` – Images used on the site

These assets combine to form a simple static site that lists curated links.
